# em:math/sign

Get the sign of a number

## Parameters

| Objective | Player/Selector | Comment |
| --------- | --------------- | ------- |
| em        | math_in1        | number  |

## Returns

| Objective | Player/Selector | Comment                                                                   |
| --------- | --------------- | ------------------------------------------------------------------------- |
| em        | math_out        | sign: -1 if number is negative, 0 if number is 0, 1 if number is positive |
