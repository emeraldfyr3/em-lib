# em:math/power

Raise a number to a power

## Parameters

| Objective | Player/Selector | Comment  |
| --------- | --------------- | -------- |
| em        | math_in1        | base     |
| em        | math_in2        | exponent |

## Returns

| Objective | Player/Selector | Comment                       |
| --------- | --------------- | ----------------------------- |
| em        | math_out        | base to the power of exponent |
